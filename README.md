# CSCI3100
2022Spring CSCI3100 Project Group G2 


For Develop Purpose:

  From GitHub to local repository:  
  $ git remote add <name> <url>  
  $ git pull  

  Initialize the package:  
  npm install  
  
  Run the app:  
  npm start  
  
  To Update File   
  $ git add .  
  $ git commit -m <comment>  
  $ git push <repository> <branch>
